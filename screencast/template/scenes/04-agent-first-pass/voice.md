After the script is ready, the agent creates one folder per scene under screencast scenes. For each scene, it writes voice dot md for narration and slide dot md for the visual fallback. If the scene needs a terminal demo, it writes a VHS tape and, when commands need verification, a vhs dot sh script that runs before recording. If a scene eventually needs a real screen recording, the agent still creates a placeholder slide in the first pass, so the whole screencast remains renderable.
