Under the hood, the workflow is mechanical. Each scene is rendered independently. A slide scene becomes slide dot md dot mp4. A terminal scene becomes VHS tape dot mp4, and if a vhs dot sh verifier exists, it runs before recording so command failures are caught early. Narration becomes voice dot md dot mp3. After each scene has visual media and optional audio, the stitcher reads the scene folders in order and creates the final MP4. This is the same pipeline whether a human runs it manually or an AI agent runs it autonomously.
