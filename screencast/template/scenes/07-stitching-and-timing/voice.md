The stitcher chooses one visual source per scene. A real video dot mp4 wins. If that is not present, it looks for another scene local video. If there is a terminal tape, it uses VHS tape dot mp4. Otherwise it falls back to the rendered slide. Scene duration is calculated mechanically: the scene lasts as long as the longer input, voiceover or visual media. If narration is longer, the final visual frame is frozen. If video is longer, audio is padded with silence.
