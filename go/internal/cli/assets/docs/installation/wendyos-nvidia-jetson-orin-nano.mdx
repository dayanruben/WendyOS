title: NVIDIA Jetson Orin Nano
description: Install WendyOS on an NVIDIA Jetson Orin Nano
---

The Jetson Orin Nano installation flow writes WendyOS to an NVMe SSD, then boots the device from that drive.

## What You Need

- NVIDIA Jetson Orin Nano Developer Kit
- NVMe SSD
- NVMe SSD to USB-C adapter
- Wendy CLI installed on your development machine

Prefer video? Watch the [Jetson Orin Nano installation walkthrough](https://youtu.be/TtDJAtBUEZI).

## Installation Steps

<Steps>
  <Step>
    ### Prepare the NVMe SSD

    Connect the NVMe SSD to an NVMe SSD to USB-C adapter, then connect the adapter to your computer.

    <video width="100%" autoPlay loop muted playsInline>
      <source src="/videos/installation/jetson-nvme-mac-adapter.mp4" type="video/mp4" />
      Your browser does not support the video tag.
    </video>

    On macOS, allow the adapter when the permission dialog appears.
  </Step>

  <Step>
    ### Install WendyOS

    Run the installer:

    ```bash
    wendy install
    ```

    <CliShot flow="os-install" step="select-device-type" alt="The Wendy CLI prompting you to choose a device type before flashing WendyOS" />

    Select **Jetson Orin Nano**, then select the attached NVMe drive. WendyOS downloads to the drive, and you'll enter your administrator password to write the image.

    The CLI also offers optional first-boot provisioning for WiFi, critical updates, and Wendy Cloud enrollment. See [`wendy install`](/docs/advanced/clients/wendy-cli/commands/os/install) for more.
  </Step>

  <Step>
    ### Install the NVMe SSD

    Remove the NVMe SSD from your computer and insert it into the Jetson's M.2 slot.

    <video width="100%" autoPlay loop muted playsInline>
      <source src="/videos/installation/jetson-nvme-bottom-insertion.mp4" type="video/mp4" />
      Your browser does not support the video tag.
    </video>
  </Step>

  <Step>
    ### Connect and Power On

    Connect the Jetson to your computer using the data-capable USB-C port, then connect the supplied power supply to a power outlet and the Jetson.

    <video width="100%" autoPlay loop muted playsInline>
      <source src="/videos/installation/jetson-mac-usb-cable.mp4" type="video/mp4" />
      Your browser does not support the video tag.
    </video>

    <video width="100%" autoPlay loop muted playsInline>
      <source src="/videos/installation/jetson-power-cable-insertion.mp4" type="video/mp4" />
      Your browser does not support the video tag.
    </video>
  </Step>

  <Step>
    ### Discover Your Device

    Wait 1-2 minutes for WendyOS to boot, then run:

    ```bash
    wendy discover
    ```

    <CliShot flow="discover" step="devices" alt="The Wendy CLI listing discovered WendyOS devices on the network" />

    Your Jetson is ready for development. See [Discovering Devices](/docs/device/discovering-devices) to learn more.
  </Step>
</Steps>
